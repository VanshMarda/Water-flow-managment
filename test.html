INTRODUCTION





A flow control provides good water management. By including a flow control in your system, you will be saving water which will ultimately save you money. A win for you and the environment!!.

The idea of this project is to develop a water flow monitoring system to overcome above problems by setting up components for detecting leakage, pipe bursts and liquid quality to ensure minimum wastage of good quality water.



This Monitoring Device and Application thus achieves the following :


1) Displays current water flow in a pipe

2) Raises alarm if water flow exceeds limit

3) Records data of water flow and water consumption in a graph for further applications

4) Monitors amount of water consumed per day

5) Calculates monthly water bills

IMPLEMENTATION APPROACH



STEP:1


The flow sensor will be installed in a pipe. The sensor will output the flow rate in pulses/min. This data is uploaded on this website via OneM2M which can be displayed by the users as well as analysts.

STEP:2


The website will display using graphs the data collected from oneM2M.

STEP:3


The data collected will be analyzed for various further applications.

STEP:4


The sensor, interfaced with the ESP32 will be set up in a 3D printed box to finish setting up the Monitoring Device.
