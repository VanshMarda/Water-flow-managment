DATA ANALYSIS





1)The sensor outputs flow rate in pulses/min



1)By counting the pulses from the output of the magnetic hall effect sensor(integrated),we can easily calculate water flow. Each pulse is approximately 2.25 milliliters.



2)The difference in flow speeds over an interval of time can be used to calculate the average consumption of water.



3)The flow rate can be used to calculate water bills in domestic as well as industrial applications.



4)Per day water consumption and average water flow will be monitored, and an alert generated if data exceeds limit

IMPLEMENTATION APPROACH



STEP:1


The flow sensor will be installed in a pipe. The sensor will output the flow rate in pulses/min. This data is uploaded on this website via OneM2M which can be displayed by the users as well as analysts.

STEP:2


The website will display using graphs the data collected from oneM2M.

STEP:3


The data collected will be analyzed for various further applications.

STEP:4


The sensor, interfaced with the ESP32 will be set up in a 3D printed box to finish setting up the Monitoring Device.
